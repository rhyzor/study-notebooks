## 🧠 Что здесь?

Подборка моих **Jupyter‑ноутбуков** для экспериментов и обучения.  
Я тестирую идеи, разбираю концепции и документирую процесс.

---

## 🚀 Быстрый старт

```bash
git clone https://github.com/rhyzor/study-notebooks.git
cd study-notebooks
```
<details>
<summary>macOS / Ubuntu</summary>

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install --upgrade pip
pip install -r requirements.txt
jupyter notebook
```
</details>

<details>
<summary>Windows (PowerShell)</summary>

```powershell
python -m venv .venv
.venv\Scripts\activate
python -m pip install --upgrade pip
pip install -r requirements.txt
jupyter notebook
```
</details>
